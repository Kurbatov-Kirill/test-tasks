## 1. Сортировка жидкостей
> Программа на Java, выполняющая поиск решения головоломки по задаваемым условиям.
>

## Использование
Находясь в папке java (testFirst/src/main/java) выполнить в терминале команду:

`java -cp . com.github.kurbatov.sorter.Main`

Программа задаст вопрос, нужно ли использовать условия задачи как входные данные?

<img width="1146" height="129" alt="image" src="https://github.com/user-attachments/assets/c3959975-d85d-4221-905a-3f90973b5fc8" />

<br>
<br>
<br>

- Если да, то количество пробирок, жидкостей и порядок капель будет автоматически подтянут в соответствии с условиями задачи, сразу переходя к выполнению поиска решения
- Если нет, то программа будет ожидать ввод необходимых данных: **`объём одной пробирки`** и **`количество видов жидкостей`**, то есть, цветов.

После этого, программа выведет стартовый набор данных


<img width="475" height="129" alt="image" src="https://github.com/user-attachments/assets/fddb7874-f880-4291-8daf-4cf196c7c5de" />


<br>
<br>
<br>

Количество пробирок рассчитывается автоматически, исходя из логики **`количество видов жидкостей + 2`**

Далее следует последовательный ввод в каждую пробирку набора капель с цветами от `1` до **`количества видов жидкостей`**, что в данном примере составляет 6

<img width="471" height="373" alt="image" src="https://github.com/user-attachments/assets/f70ea4c7-9640-45ef-bdd5-a3161bda1625" />

<br>
<br>
<br>
